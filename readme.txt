

1. At first you have to register for admin.

data

    {
	"userName": "as your wish",
	"phone": "as your wish",

    }


2. Admin login route is '/adminLogin'


3. You have to create a customer with name 'guest' to track the guests order. just go to registration page and register as user name 'guest'. Name must be guest. Address, phone number whatever you like.
